# Public Assets

This directory contains static assets for the Resume Builder application.

## Directory Structure

```
public/
├── images/
│   └── templates/      # Template preview images
├── fonts/              # Custom fonts (if any)
├── assets/             # Other static assets (logos, icons, etc.)
└── README.md
```

## Usage

- **images/templates/** - Store template preview screenshots for the homepage
- **fonts/** - Custom fonts that aren't available via CDN
- **assets/** - General static assets like logos, favicons, etc.

## Notes

- All files in this directory are publicly accessible
- Images should be optimized for web (use WebP when possible)
- Use descriptive file names (e.g., `modern-template-preview.webp`)
