# Debug Issue

Help debug an issue in the codebase.

## Arguments

- `$ARGUMENTS` - Description of the issue or error message

## Steps

1. Search the codebase for related code using the error message or description
2. Identify potential root causes
3. Check related files and dependencies
4. Propose a fix with explanation
5. If needed, suggest debugging steps or console.log placements

Focus on understanding the data flow and state management patterns used in this project.
